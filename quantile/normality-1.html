Test normality of a variable in Stata

In Stata, we can test normality of a distribution by either graphical or numerical methods.

Graphical methods
Command Plot type
stem var stem-and-leaf
dotplot var scatterplot
graph box var box plot
histogram var, kdensity normal histogram
kdensity var, normal kernel density
pnorm var p-p plot
qnorm var q-q plot

Numerical methods
Command Test
swilk var Shapiro-Wilk
sfrancia var Shapiro-Francia
sktest var Skewness/Kurtosis

→Be aware that in these tests, the null hypothesis states that the variable is normally distributed.

→when skewness is 0, the distribution is symmetric; when kurtosis is 3, it is normal.
